Landslide Detection using Deep Learning
This project focuses on detecting landslide-affected regions from images using deep learning-based image segmentation techniques. The goal is to assist in early detection and risk analysis for disaster management.

Overview
Landslides are a major natural hazard causing damage to life and infrastructure. This project uses computer vision and deep learning models to automatically identify landslide regions from satellite or terrain images.

Model & Approach
- Architecture: Attention U-Net / U-Net (customizable)
- Task: Image Segmentation
- Loss Functions: Dice Loss + Binary Cross Entropy
- Metrics: Dice Score, IoU, Accuracy, Precision, Recall

 Tech Stack
- Python 
- PyTorch 
- OpenCV
- NumPy / Pandas
- Matplotlib
